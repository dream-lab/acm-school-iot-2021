====================OS Requirements=========================
Ubuntu 20.04 (focal) / Ubuntu 18.04 (bionic)


====================ROS Installation========================
==> For Ubuntu 20.04 users, follow ROS noetic installation steps: http://wiki.ros.org/noetic/Installation/Ubuntu 





==> For Ubuntu 18.04 users, follow ROS kinetic installation steps: http://wiki.ros.org/melodic/Installation/Ubuntu 

sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list' 

curl -s https://raw.githubusercontent.com/ros/rosdistro/master/ros.asc | sudo apt-key add -

sudo apt update 

sudo apt install ros-melodic-desktop-full 

echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc
source ~/.bashrc

sudo apt install python-rosdep python-rosinstall python-rosinstall-generator python-wstool build-essential

sudo apt install python-rosdep 

sudo rosdep init
rosdep update 


====================Setting up the launch Files================

cd /acm-school-iot-2021/Gazebo/acm_tutorial_gazebo

Open the new terminals in the same window
==> Terminal 1: Starts the ros master
source /opt/ros/melodic/setup.bash 
catkin_make 
roscore 

==> Terminal 2: Starts the gazebo world 
source devel/setup.bash 
roslaunch rotors_gazebo gazebo_launch.launch

==> Terminal 3: Launches the drone. Uncomment lines 26-44 for launching 2nd drone. 
source devel/setup.bash 
roslaunch rotors_gazebo mav_acm.launch

==> Terminal 4: Starts subscribing the drone's position. Uncomment line 19 for 2nd drone. 
source devel/setup.bash  
python test_subscriber.py

==> Terminal 5: Starts publishing the waypoint to the drone. Uncomment lines 88, 91 for 2nd drone. 
source devel/setup.bash  
python test_publisher.py
